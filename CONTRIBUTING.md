# Katkıda Bulunma Rehberi

Bu projeye katkıda bulunmak için teşekkür ederiz! 🎉

## Nasıl Katkıda Bulunabilirim?

1. Bu repoyu **fork** edin
2. Yeni bir **branch** oluşturun: `git checkout -b feature/yeni-ozellik`
3. Değişikliklerinizi **commit** edin: `git commit -m 'Yeni özellik eklendi'`
4. Branch'inizi **push** edin: `git push origin feature/yeni-ozellik`
5. Bir **Pull Request** açın

## Kod Standartları

- HTML5 semantik etiketleri kullanın
- CSS Custom Properties (variables) kullanın
- Vanilla JavaScript tercih edin
- Responsive tasarım prensiplerine uygun olun

## Raporlama

Bir hata bulduysanız veya özellik öneriniz varsa, lütfen [Issues](https://github.com/kullaniciadi/morse-alfabesi/issues) bölümünü kullanın.
